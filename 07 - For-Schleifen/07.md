Wandeln Sie den gegebenen *for*-Schleifen in gleichartige *while*-Schleifen um.

```cpp
#include <stdio.h>

int main() {
  int result = 0;
  int max = 0;
  scanf("%i", &max);
  
  for(int i = 0; i<max; i+=3) {
    result += i;
  }
  
  for(int u = 1; u<max; u++) {
    result--;
  }

  printf("%i",result);
  return 0;
}
```
